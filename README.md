# calculator-versions

Interactive calculator project that shows the evolution of features across four versions:

- Version 1.0: basic arithmetic
- Version 2.0: advanced operations
- Version 3.0: memory and history
- Version 4.0: scientific calculator

## Run the web interface

From this folder, start a simple local server:

```powershell
python -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

The web app files are:

- `index.html`
- `style.css`
- `script.js`
